Menghitung Angka Metabolisme Basal

Angka Metabolisme Basal

Digunakan untuk menghitung kebutuhan energi dasar tubuh (kalori minimal) yang dibutuhkan seseorang dalam kondisi istirahat total.

Cara menghitung angka metabolisme :
Untuk laki-laki : (10 x Berat Badan) + (6.25 x Tinggi Badan) - (5 x Umur) + 5
Untuk Perempuan : (10 x Berat Badan) + (6.25 x Tinggi Badan) - (5 x Umur) - 161
