Indeks Massa Tubuh

Indeks Massa Tubuh (IMT) adalah cara cepat untuk mengetahui apakah berat badanmu ideal berdasarkan tinggi badan. Cocok untuk cek status gizi secara umum.

Cara Menghitung IMT :
Berat badan (kg) / Tinggi badan (m)2
